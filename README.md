# Reinhardt's Personal Hub

## Local Setup

note that you must have jekyll installed locally
```
bundle install
```

## Run the Server

Run the following command within the Command Line Interface
```
jekyll serve
```

## View the Application

go to one of the following url
```
localhost:4000
127.0.0.1:4000
```
